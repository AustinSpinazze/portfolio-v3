# Personal Portfolio v3

This personal Portfolio Project was built with NextJS, Tailwind, HeadlessUI, MDX, Sanity, and Feed. It was inspired by the Tailwind Spotlight UI template, but with its own twist and improvements.

## Getting started

To get started with this template, first install the npm dependencies:

```bash
npm install
```

Next, run the development server:

```bash
npm run dev
```

Finally, open [http://localhost:3000](http://localhost:3000) in your browser to view the website.
